# Hydrogen.Application

Application framework for Hydrogen-based apps. It provides lifecycle management, dependency injection, settings, logging, and command-line support.

## Overview
Hydrogen.Application supplies shared application infrastructure used across console, desktop, and service-style apps in this repo.

## Key features
- Application lifecycle hooks
- DI integration
- Command-line parsing helpers
- Settings and configuration utilities
- Logging and diagnostics helpers
