# Hydrogen.DApp.Host

Host process for running Hydrogen DApp nodes as standalone services.

## Overview
Launches and manages `Hydrogen.DApp.Node`, handling process lifecycle, logging, and shutdown.

## Usage

```bash
dotnet Hydrogen.DApp.Host.dll
```
