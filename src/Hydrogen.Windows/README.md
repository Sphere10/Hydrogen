# Hydrogen.Windows

Windows-specific utilities and platform integration for Hydrogen applications.

## Overview
Provides helpers for registry access, Windows services, event logging, and security-related APIs.

## Key features
- Windows security and SID helpers
- Registry access utilities
- Windows service management helpers
- Event log integration

## Permissions
Many operations require Administrator privileges. Reading system information typically does not.
